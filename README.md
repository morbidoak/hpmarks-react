## Download & Run locally

Install npm, if you haven't. For example:
```bash
curl https://npmjs.org/install.sh | sh
```
clone this repository:
```bash
cd /path/to/projects
git clone https://github.com/morbidoak/hpmarks-react
```
go to it:
```bash
cd hpmarks-react
```
install dependencies:
```bash
npm install
```
and run:
```bash
npm run
```
